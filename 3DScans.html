Regard 3D Scans

Regard 3D is an open-source photogrammetry software that works on systems that do not have a Nvidia GPU. It can be downloaded here. I like it because each step of the photogrammetry process is done seperately, so you can see the final product develop. However, each step must be begun manually, so it requires a higher level of supervision than Meshroom and other photogrammetry softwares.

You can see some of scans on the Instagram.
